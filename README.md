# RaspberryPiSt7789nLED
240x240 IPS LCD Display Run with Raspberry Pi. reference https://www.mikan-tech.net/entry/raspi-st7789-lcd
* I use Raspberry Pi 4 Model B 8GB
* LCD ... SPI communication, st7789 driver
